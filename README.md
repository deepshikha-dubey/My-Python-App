# My-Python-App
Built a Python Flask application and deployed it using Docker

 Created a Python Flask web application and containerized it using Docker.
Enhanced the UI and added dynamic data like hostname and time to demonstrate container behavior.
created a CI/CD pipeline using GitHub Actions with secure credentials
pushed the image to Docker Hub, (docker tag my-python-app deepshikhadubey/my-python-app), docker push deepshikhadubey/my-python-app
and deployed it on Render : https://my-python-app-le4m.onrender.com/
